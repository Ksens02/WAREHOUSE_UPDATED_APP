# warehouse_app
This web application is a full-stack inventory management dashboard built to manage a sports-related product warehouse. Connected to a relational database, it provides a clean, professional interface for viewing and managing all warehouse items — including product names, prices, stock quantities, shoe sizes, and other relevant attributes. The dashboard is designed for efficient day-to-day inventory oversight, with all item data displayed in an organized, easy-to-navigate layout.
Admin users have full control over the inventory through built-in tools to add new products and remove existing ones, keeping the warehouse records accurate and up to date. The application is built with scalability and usability in mind, making it straightforward for warehouse staff or administrators to maintain a complete and current catalog of sports merchandise.
